# Proxy Checker

Проверяет доступность прокси-серверов на вашей машине и сортирует их по категориям работоспособности.

##  Поддерживаемые протоколы

| Протокол | Особенности |
|----------|-------------|
| **VLESS** | Reality, TLS, WebSocket, gRPC |
| **VMESS** | ws, tcp, grpc + TLS |
| **Trojan** | ws, tcp, grpc + TLS |
| **Hysteria2** | с обфускацией |
| **Shadowsocks** | AES-GCM, ChaCha20, 2022-Blake3 |

##  Безопасность

Автоматически отфильтровывает:
-  Прокси с `insecure=1` без SNI (MITM-уязвимость)
-  SSRF-адреса (`127.0.0.1`, `192.168.x.x`, `10.x.x.x`)
-  Слабые шифры Shadowsocks (`rc4`, `table`, `none`)
-  Невалидные UUID, порты, пароли
-  Огромные конфиги (защита от DoS)

##  Установка

### Требования
- **Python 3.8+**
- **Xray-core** (https://github.com/XTLS/Xray-core/releases/latest)

```bash
pip install --upgrade pip >nul 2>&1
pip install -r requirements.txt
и запустить run.bat
```

В конфигах есть предустановленые ссылки на открытые источники на гитхаб для теста. Также можно загрузить из файла.
